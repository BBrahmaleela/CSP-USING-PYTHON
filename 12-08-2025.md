PROBLEM STATEMENT 2:
Write a program to print a given number is prime number or not?
n = int(input())
count = 0
for i in range(1, n + 1):
    if n % i == 0:
        count += 1
if count == 2:
    print("yes, it's a prime number")
else:
    print("no, it's not a prime number")
